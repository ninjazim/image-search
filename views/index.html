Image Search Abstraction Layer

  • Searching for images:
  • Pass a string as a parameter to /api/imagesearch/:query
  • Example: /api/imagesearch/pancakes
  • Add an offset paramater with a number to see a different page of results
  • Example: /api/imagesearch/pancakes?offset=2
  • Viewing recent searches:
  • To see a list of the 10 most recent searches, navigate to /api/imagesearch/recent
